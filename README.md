## Project Tasks / Implementation

- Deployed EC2 instances in **two different AWS regions** using the same Terraform resource configuration.
- Configured **remote Terraform backend using an S3 bucket** for centralized state management.
- Hosted a **static web page on EC2 instances using the Nginx web server**.
- Mounted an **S3 bucket to the EC2 instances using IAM roles and policies** to allow secure access.
- Implemented **cross-region replication between two S3 buckets** located in different AWS regions.
- Automated **Nginx installation and static webpage deployment using Terraform null_resource and provisioners**.
